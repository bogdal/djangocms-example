=================
djangocms-example
=================

Example project using Django-CMS 3 (dev)

Quickstart
==========================

::

 git clone https://github.com/bogdal/djangocms-example.git your-project
 cd your-project
 make init
 

Configure environment variables:

.. code-block:: bash

    export DEBUG='on'
    export SECRET_KEY='...'
